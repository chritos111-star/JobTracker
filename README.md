# JobTracker

Windows Forms alkalmazás állásjelentkezések nyilvántartására.

A projekt célja egy egyszerű asztali alkalmazás készítése, amely segít az álláskeresési folyamat követésében. 
A felhasználó cégeket és állásjelentkezéseket kezelhet adatbázis segítségével.

## Funkciók

- Cégek kezelése (CRUD műveletek)
- Állásjelentkezések kezelése (CRUD műveletek)
- Cégek és jelentkezések kapcsolata
- Jelentkezési státuszok kezelése
- Adatok módosítása és törlése
- Adatellenőrzés (validáció)
- Adatok megjelenítése DataGridView segítségével

## Technológiák
